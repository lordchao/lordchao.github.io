Chapter8

This chapter is mainly talk about graphical model, graphical model can show us that some sample distribution like exponential family are lined together. This part is so import that it can link the whole book.

Import concept

  • Bayesian Network: directed graph models, it is useful for expressing causal relationships between random variables.
  • Markov Random field: undirected graph models, it is better for expressing soft constraints between random variables.
  • Conditional Independence: $p(a|b,c)=p(a|c)$, that is a is conditionally independent of b given c. This concept is very import for simplify the graphic model.

Bayesian Network

discrete variable can be used to

Two cases are particularly worthy of note, namely when the parent and child node each correspond to discrete variables and when they each correspond to Gaussian variables, because in these two cases the relationship can be extended hierarchically to construct arbitrarily complex directed acyclic graphs. We begin by examining the discrete case.

In the Bayesian Network we can use the property of conditional independence to factorize the join distribution and many problems can be expressed in this way such as inferring the mean of a univariate Gaussian distributions and naive bayes.

We can view a graphical model (in this case a directed graph) as a filter in which a probability distribution $p(x)$ is allowed through the filter if, and only if, it satisfies the directed factorization property. The set of all possible probability distributions $p(x)$ that pass through the filter is denoted as Direct Factorization

Markov Random Fields

For the undirected graphic models, we also define the concept of conditional independence, but we use a set of nodes instead of a single node. And therefore we also have the rule for the factorization of undirected graphical models.
Note that this is exactly the same as the d-separation criterion except that there is no ‘explaining away’ phenomenon. Testing for conditional independence in undirected graphs is therefore simpler than in directed graphs.
The conditional independence property can be expressed as follow:
$$p(x_i,x_j|x_{/{x_i,x_j}})=p(x_i|x_{/{x_i,x_j}})p(x_j|x_{/{x_i,x_j}})$$
where $x_{/{x_i,x_j}}$ denotes the set $x$ of all variables with $x_i$ and $x_j$ removed.
Due to we need a subset of the graph, so we define clique to represent a subset of nodes in which each node is fully connected. And We can therefore define the factors in the decomposition of the joint distribution to be functions of the variables in the maximal cliques.
The joint distribution is written as the product of potential functions $\psi_C(x_C)$ over the maximal cliques of the graph.
$$p(x)=\frac{1}{Z}\prod_C\psi_C(x_C)$$
Here the quantity Z, sometimes called the partition function, is a normalization constant and is given by
$$Z=\sum_X\prod_C\psi_C(x_C)$$
Here comes to the very interesting concept in the deep learning - energy function. Because the potential function are strictly positive, we can express it in exponential.
$$\psi_C(x_C)=\exp{-E(x_C)}$$
where $E(x_C)$ is so called energy function.
There is no restrict of the potential function which is different from directed graphic models, so we can choose the potential function that give us both flexibility and difficulty.
