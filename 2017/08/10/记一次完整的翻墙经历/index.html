记一次完整的翻墙经历

Posted on 2017-08-10 Edited on 2019-10-09 In Technique Disqus:

首先先在vultr上买了一个日本的vps，然后利用这个vps搭建ss服务器。
成功购买vps之后，会得到一个ipv4的公网地址，然后通过ssh连接到vps搭建shadowsocks.

server端

第一步安装shadowsocks:
yum install python-setuptools m2crypto
easy_install pip
pip install shadowsocks
shadowsock是个sock5代理，所以需要server端和client端，首先创建在server端的json文件
vi /etc/shadowsocks.json
内容为：

这里需要注意！这里仅仅开启了一个端口(443端口)，也就意味着只能有一个client连接到server，如果手机连了，那电脑就连不上了,如果需要多个用户，就需要配置多个端口。

接下来需要注意要在防火墙中开启并重置端口，也就是要这些端口对外开放，否则是连不上的。
firewall-cmd –zone=public –add-port=8080/tcp –permanent
firewall-cmd –zone=public –add-port=8081/tcp –permanent
firewall-cmd –reload
打开fast-open
vim /etc/rc.local
在文末添加
echo 3 > /proc/sys/net/ipv4/tcp_fastopen
退出编辑系统控制文件
vim /etc/sysctl.conf
在最后添加
net.ipv4.tcp_fastopen = 3
最后将json配置文件中的fast-open打开
“fast_open”:true
这样shadowsocks的服务端就配置好了，但是连上的话速度会有些慢，看视频的话会很卡，因此这里我们需要安装锐速加速器
首先需要更改centos的内核
rpm -ivh http://soft.91yun.org/ISO/Linux/CentOS/kernel/kernel-3.10.0-229.1.2.el7.x86_64.rpm –force
查看内核是否更换成功
rpm -qa | grep kernel
重启
sudo reboot
使用脚本安装锐速
wget -N –no-check-certificate https://raw.githubusercontent.com/91yun/serverspeeder/master/serverspeeder-all.sh && bash serverspeeder-all.sh
配置文件
vim /serverspeeder/etc/config
将advinacc的0改为1
开启ss服务端
sudo ssserver -c /etc/shadowsocks.json -d start
开启加速
/serverspeeder/bin/serverSpeeder.sh start

但是每次输入命令会很麻烦因此需要安装supervisor
sudo apt-get install supervisor
配置conf文件
sudo vim /etc/supervisor/supervisord.conf
在文末加入下面文档

启动supervisor
service supervisor restart
设置开机自启动
sudo vim /etc/rc.local
加入
service supervisor start

配置浏览器

所有东西设置好之后还差最后一部就可以在电脑上上网了，使用chrome浏览器下载switchyomega代理软件，将代理设置为本机127.0.0.1端口设置为1080即可上网！
